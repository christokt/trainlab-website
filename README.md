# TRAIN Lab Modern GitHub Pages Site

Upload all files in this folder directly to your GitHub repository root. Do not upload the ZIP itself.

## Enable GitHub Pages
1. Repository → Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: / (root)
5. Save

## Editing
- Homepage: `index.html`
- Research page: `research.html`
- Team page: `team.html`
- Publications page: `publications.html`
- Main design: `style.css`

## Custom domain
In GitHub Pages, add your custom domain, then update GoDaddy DNS according to GitHub Pages instructions.
